# Website-Backend

API for usine-du-futur project

```
git clone https://github.com/usine-du-futur-enceinte/Website-Backend.git
```
```
cd Website-Backend
```
```
npm install
```

# Run the server
```
npm run dev
```

# Environment variables
```
PORT=3030
MONGO_URI=mongodb+srv://[DB_USERNAME]:[DB_PASSWORD]@usinedufutur.ltjag.mongodb.net/
```
